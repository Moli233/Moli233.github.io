java的基础语法以及概念 1

一个JAVA程序，可看为是一系列对象的集合，而这些对象通过调用彼此的方法互相协同工作

而”她”有一下几点关键的概念

想要玩弄她，就得了解她

概念:

她是一个怎么样的女人

  • 对象(Object)：对象是类的一个实例，它有状态和行为，例如我定义一个类，叫傻逼，
    • 它的状态有很多:名字 品种 性别
    • 它的行为有很多: 人傻钱多 事后狂怒 情绪失控
  • 类(class):一个模板，描述一类对象的状态与行为
  • 方法:即行为，一个类可以有很多方法。逻辑运算，数据修改，以及所有动作都是方法中完成的
    所以很重要啊!
  • 实例变量:每个对象都有独特的实例变量，对象的状态由这些实例变量的值决定
    主人的命令罢了

华丽分割线华而不实

Java 标识符:

Java 所有的组成部分都需要名字。类名、变量名以及方法名都被称为标识符。

关于 Java标识符，有以下几点需要注意：

所有的标识符都应该以字母（A-Z 或者 a-z）,美元符（$）、或者下划线（_）开始

首字符之后可以是字母（A-Z 或者 a-z）,美元符（$）、下划线（_）或数字的任何字符组合

关键字不能用作标识符

标识符是大小写敏感的

真敏感呢

合法标识符举例：age、$salary、_value、__1_value
