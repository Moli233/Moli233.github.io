C#的数据类型

  • 整数类型
  1. byte(占用一个字节，也就是8个bit)：范围是 0到255 也就是 0到2^8-1与JAVA不同JAVA的byte型是-128~127
  2. short(占用两个字节，也就是16个bit)：范围是 -32768到32767 也就是 -2^15到2^15-1
  3. int(占用四个字节，也就是32个bit)：范围是-2147483648到2147483647 也就是-2^31到2^31-1
  4. long(占用八个字节也就是64个bit)：范围是-1587450786638754354到1587450786638754353 也就是-2^63到2^63-1,并且long型赋值的结尾要加大写或小写的L表名为long型，如果赋值未大于int型范围，也可不加

  • 浮点型

    1. float(占用四个字节，也就是32个bit)：范围是-3.4 x 10^38 到 + 3.4 x 10^38 ，float型赋值的结尾需要加大写或小写的F float是32位单精度
    2. double型(占用八个字节，也就是64个bit): 范围是(+/-)5.0 x 10^-324 到 (+/-)1.7 x 10^308总之非常大
    3. 浮点型里只有float转double虽然有点废话，但还是得说

  • 十进制型

  1. decimal的范围是(-7.9 x 10^28 到 7.9 x 10^28) / 10^0~28 decimal 是128 位精确的十进制值，有28-29 有效位数，且赋值得加大写或小写M
  • 布尔型
  1. bool的范围只有 True or false 一般用于条件判断
  • 字符型

  1. string 字符串，记得””

  2. char 单字符，记得’’

  • 空类型
  1. null即为空值整数型和bool型还有char型是不能直接用null的
